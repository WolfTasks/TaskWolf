### Task 2: Localize ReportsPage

**Files:**

- Modify: `frontend/scripts/i18n-allowlist.json`

**Interfaces:**
- Consumes: `reports:title`, `reports:burndown.*`, `reports:velocity.*` (Task 1).

- [ ] **Step 1: Add the hook import**

```ts

```

- [ ] **Step 2: Add the hook**

Add as the first line of the `ReportsPage` body (before `const { key } = useParams…`):

```ts

```

- [ ] **Step 3: Localize the page + burndown section**

Replace:

```tsx
        <h1 className="text-2xl font-bold">Reports</h1>
```
→ `<h1 className="text-2xl font-bold">{t('title')}</h1>`

```tsx
        <h2 className="text-lg font-semibold text-white mb-4">Burndown Chart</h2>
        {burndownData.length === 0 ? (
          <p className="text-gray-500 text-sm">No burndown data available. Sprint must have a start date and story points.</p>
```

with:

```tsx
        <h2 className="text-lg font-semibold text-white mb-4">{t('burndown.title')}</h2>
        {burndownData.length === 0 ? (
          <p className="text-gray-500 text-sm">{t('burndown.empty')}</p>
```

Add `name` to the burndown `<Line>`s:

```tsx

```

- [ ] **Step 4: Localize the velocity section**
